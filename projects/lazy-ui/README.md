# Lazy Ui

A library of Angular Material reusable components

### Components

**Inputs**
* Generic Input Field
* Numbers Input Field
* File Upload Field: Contains box area and button (set [style] attribute to either 'upload-button' or 'upload-area')

**Buttons**
* Generic Button
* Mat Stepper Buttons (Next and Back)

**Directives**
* Phone Number
* Numbers
* Email
